# افزونه وردپرس اپ مشخصات محصول

برای نصب ساده، فایل زیر را در وردپرس بارگذاری کنید:

```text
product-specs-api.zip
```

بعد از فعال‌سازی:

```text
تنظیمات → API مشخصات محصول
```

توکن اتصال را کپی کنید و داخل اپلیکیشن وارد کنید.


## ذخیره محتوا

این افزونه علاوه بر ذخیره ویژگی‌های اختصاصی محصول، مقدار `product_content` را هم دریافت می‌کند و آن را در توضیح کوتاه محصول ووکامرس ذخیره می‌کند.
