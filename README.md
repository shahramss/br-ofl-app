# اپلیکیشن اپ مشخصات محصول - ثبت مشخصات فنی محصولات ووکامرس

این پروژه شامل دو بخش است:

1. اپلیکیشن موبایل Flutter برای اندروید
2. افزونه وردپرس برای اتصال امن به ووکامرس و ذخیره مشخصات فنی روی محصول

اپلیکیشن از دو سرویس هوش مصنوعی پشتیبانی می‌کند:

- ChatGPT / OpenAI
- DeepSeek

کلیدهای هوش مصنوعی داخل خود اپلیکیشن وارد می‌شوند و روی وردپرس ذخیره نمی‌شوند. این برای سایت‌هایی مناسب است که سرور وردپرس داخل ایران است و ارتباط مستقیم سرور با AI مشکل دارد.

---

## ساختار پوشه‌ها

```text
product_specs_project/
├── mobile_app/                  سورس اپلیکیشن Flutter

├── .github/workflows/            فایل ساخت APK با GitHub Actions
├── docs/                         مستندات کمکی
└── README.md                     همین راهنما
```

---

## مرحله ۱: نصب افزونه وردپرس

داخل فایل ZIP این مسیر را پیدا کنید:

```text
wordpress_plugin/product-specs-api
```

برای نصب افزونه دو راه دارید.

### راه ساده‌تر

یک فایل آماده افزونه هم داخل خروجی قرار داده شده:

```text
product-specs-api.zip
```

در وردپرس بروید به:

```text
پیشخوان وردپرس
→ افزونه‌ها
→ افزودن
→ بارگذاری افزونه
```

فایل `product-specs-api.zip` را آپلود و فعال کنید.

### بعد از فعال‌سازی افزونه

بروید به:

```text
پیشخوان وردپرس
→ تنظیمات
→ API مشخصات محصول
```

توکن اتصال را کپی کنید. این توکن را باید داخل اپلیکیشن وارد کنید.

---

## مرحله ۲: آپلود پروژه در GitHub

1. فایل ZIP اصلی پروژه را باز کنید.
2. وارد پوشه `product_specs_project` شوید.
3. یک ریپازیتوری جدید در GitHub بسازید.
4. همه فایل‌ها و پوشه‌های پروژه را داخل ریپازیتوری آپلود کنید.
5. حتماً پوشه `.github` هم آپلود شود، چون فایل ساخت APK داخل آن است.

---

## مرحله ۳: گرفتن خروجی APK از GitHub Actions

بعد از آپلود پروژه:

1. وارد ریپازیتوری GitHub شوید.
2. از بالا روی تب `Actions` کلیک کنید.
3. اگر GitHub اجازه خواست، Actions را فعال کنید.
4. روی workflow با نام `Build Android APK` کلیک کنید.
5. اگر خودش اجرا نشد، دکمه `Run workflow` را بزنید.
6. چند دقیقه صبر کنید تا build تمام شود.
7. بعد از اتمام، پایین صفحه بخش `Artifacts` را ببینید.
8. فایل زیر را دانلود کنید:

```text
product-specs-debug-apk
```

داخل آن فایل APK قرار دارد:

```text
app-debug.apk
```

این فایل را روی گوشی اندروید نصب کنید.

---

## مرحله ۴: تنظیم اپلیکیشن روی گوشی

بعد از نصب APK:

1. اپلیکیشن را باز کنید.
2. از منوی پایین وارد `تنظیمات` شوید.
3. در بخش وردپرس این موارد را وارد کنید:

```text
آدرس سایت:
https://example.com

توکن اتصال وردپرس:
توکنی که از افزونه کپی کردید
```

4. روی `تست اتصال وردپرس` بزنید.
5. اگر پیام اتصال موفق دیدید، اتصال درست است.

---

## مرحله ۵: تنظیم هوش مصنوعی داخل اپلیکیشن

در همان صفحه تنظیمات، بخش هوش مصنوعی را می‌بینید.

### برای ChatGPT / OpenAI

ارائه‌دهنده را روی `ChatGPT / OpenAI` بگذارید.

```text
OpenAI API Key:
sk-xxxxxxxx

مدل OpenAI:
gpt-4o-mini
```

### برای DeepSeek

ارائه‌دهنده را روی `DeepSeek` بگذارید.

```text
DeepSeek API Key:
xxxxxxxx

مدل DeepSeek:
deepseek-chat
```

بعد روی `ذخیره تنظیمات` بزنید.

---

## مرحله ۶: استفاده از اپلیکیشن

1. وارد صفحه `دسته‌بندی‌ها` شوید.
2. دسته‌بندی محصول را انتخاب کنید.
3. محصول مورد نظر را باز کنید.
4. مشخصات فعلی محصول را ببینید.
5. روی `شروع ضبط ویس` بزنید.
6. مشخصات محصول را فارسی بگویید.
7. متن خام ویس نمایش داده می‌شود.
8. اگر لازم بود متن خام را ویرایش کنید.
9. روی `تحلیل با هوش مصنوعی` بزنید.
10. مشخصات فنی استخراج‌شده را بررسی و ویرایش کنید.
11. محتوای یک‌پاراگرافی محصول را بررسی و ویرایش کنید. این متن بر اساس مشخصات واردشده ساخته می‌شود و در پایان آن جمله «این محصول با ارسال فوری از بازار قفل سفارش بدید» قرار می‌گیرد.
12. روش ذخیره را انتخاب کنید:

```text
افزودن به مشخصات قبلی
یا
جایگزینی مشخصات قبلی
```

حالت پیش‌فرض `افزودن به مشخصات قبلی` است.

13. روی `ارسال مشخصات` بزنید.
14. مشخصات روی همان محصول در ووکامرس ذخیره می‌شود و محتوای تولیدشده هم به توضیح کوتاه محصول اضافه می‌شود.


---

## قابلیت تولید محتوای محصول

بعد از تحلیل ویس، اپلیکیشن فقط مشخصات فنی نمی‌سازد. یک متن یک‌پاراگرافی برای همان محصول هم تولید می‌کند.

این متن:

```text
بر اساس نام محصول و مشخصات فنی گفته‌شده ساخته می‌شود
قابل ویرایش است
روی توضیح کوتاه محصول ووکامرس ذخیره می‌شود
در پایانش این جمله قرار می‌گیرد:
این محصول با ارسال فوری از بازار قفل سفارش بدید
```

اگر در ویس مشخصات بیشتری گفته شود، حتی اگر جزو فیلدهای پیش‌فرض دسته‌بندی نباشد، اپلیکیشن آن را هم به لیست مشخصات اضافه می‌کند.

---

## نکته خیلی مهم درباره ذخیره مشخصات

افزونه مشخصات را به‌صورت ویژگی اختصاصی همان محصول ذخیره می‌کند.

یعنی:

```text
is_visible = true
is_variation = false
is_taxonomy = false
```

این افزونه این کارها را انجام نمی‌دهد:

```text
Global Attribute نمی‌سازد
pa_ taxonomy نمی‌سازد
term عمومی نمی‌سازد
ویژگی را وارد فیلتر محصولات نمی‌کند
ویژگی را برای محصول متغیر فعال نمی‌کند
شورت‌کد جدید برای نمایش مشخصات نمی‌سازد
```

مشخصات فقط در بخش ویژگی‌های همان محصول نمایش داده می‌شود.

## خطاهای رایج

### خطای اتصال وردپرس

این موارد را بررسی کنید:

- آدرس سایت با `https://` وارد شده باشد.
- افزونه فعال باشد.
- ووکامرس فعال باشد.
- توکن را درست کپی کرده باشید.
- REST API وردپرس بسته نشده باشد.

### خطای هوش مصنوعی

این موارد را بررسی کنید:

- کلید API درست باشد.
- ارائه‌دهنده درست انتخاب شده باشد.
- مدل درست وارد شده باشد.
- اینترنت گوشی به OpenAI یا DeepSeek وصل باشد.
- اگر با OpenAI مشکل دارید، DeepSeek را تست کنید یا برعکس.

### ویس به متن کار نمی‌کند

این مورد به سرویس گفتار به متن گوشی بستگی دارد.

- دسترسی میکروفون را بدهید.
- زبان فارسی گوشی را فعال کنید.
- اگر تبدیل ویس کار نکرد، متن خام را دستی تایپ کنید.
