# راهنمای گرفتن APK از GitHub Actions

1. پروژه را در GitHub آپلود کنید.
2. وارد تب Actions شوید.
3. workflow با نام Build Android APK را اجرا کنید.
4. بعد از موفق شدن، از پایین صفحه Artifacts را دانلود کنید.
5. فایل app-debug.apk را روی گوشی نصب کنید.

اگر build خطا داد، صفحه همان اجرا را باز کنید و روی مرحله قرمز کلیک کنید تا متن خطا را ببینید.
