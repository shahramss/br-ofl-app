# رفع خطاهای رایج

## اتصال وردپرس برقرار نیست

- افزونه فعال باشد.
- ووکامرس فعال باشد.
- آدرس سایت با https:// وارد شود.
- توکن درست باشد.
- REST API وردپرس بسته نشده باشد.

## محصول ذخیره نمی‌شود

- محصول باید وجود داشته باشد.
- توکن باید درست باشد.
- مشخصات نباید خالی باشند.

## هوش مصنوعی کار نمی‌کند

- کلید API را بررسی کنید.
- مدل را بررسی کنید.
- اینترنت گوشی باید به OpenAI یا DeepSeek دسترسی داشته باشد.

## ویس به متن کار نمی‌کند

- دسترسی میکروفون را فعال کنید.
- زبان فارسی را روی گوشی فعال کنید.
- در صورت مشکل، متن را دستی وارد کنید.
